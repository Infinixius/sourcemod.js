connect

Fires when a player connects to the server. Make sure to fetch with Players.fetch() before interacting with players. SourceMod API Reference

Parameters:
Name Type Description
id number

ID of the player, specific to this server.
name string

Username of the player.
ip string

IP address of the player. Will be none if this player is a bot.
steamid string

Steam ID of the player, in steamID3 format (example: [U:1:1049895252]). Will be BOT if the player is a bot.
bot bool

Will be true if this player is a TFBot (or otherwise not a human player), or false if otherwise.

disconnect

Source:

Fires when a player disconnects to the server. Make sure to fetch with Players.fetch() before interacting with players. SourceMod API Reference

Parameters:
Name Type Description
id number

ID of the player, specific to this server.
name string

Username of the player.
steamid string

Steam ID of the player, in steamID3 format (example: [U:1:1049895252]). Will be BOT if the player is a bot.
reason string

Reason for the player disconnecting. (examples: Client Disconnect, Punting bot, server is hibernating, Kicked from server, etc.)
bot bool

Will be true if this player is a TFBot (or otherwise not a human player), or false if otherwise.
